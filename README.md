# Secure Password Generator using Python

A command-line Secure Password Generator built using Python. This project generates strong and customizable passwords by allowing users to choose password length and character types.

The application is designed to help users create secure passwords that are difficult to guess and suitable for protecting online accounts and personal information.

## 📌 Project Overview

This project creates random passwords using Python's built-in security libraries.

Users can customize their passwords by selecting:

* Password length
* Lowercase letters
* Uppercase letters
* Numbers
* Special symbols

The application ensures that generated passwords are randomized and secure.

Additionally, if the `pyperclip` library is installed, the generated password can automatically be copied to the clipboard.

## 🚀 Features

* Secure password generation
* Customizable password length
* Lowercase letter support
* Uppercase letter support
