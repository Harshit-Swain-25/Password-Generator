==============================
     Password Generator
==============================

Description:
A Python program that generates strong random
passwords. Users can specify the length, and
the program will create a password using
letters, numbers, and special characters.

Features:
- Generates random secure passwords
- Allows user to choose password length
- Includes uppercase, lowercase, digits, symbols
- Prevents weak passwords

How to Run:
1. Save the file as 01_main.py
2. Open VS Code terminal.
3. Run the program with:
   python3 01_main.py
